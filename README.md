# Золотое Яблоко — Коллекционные карточки

Мини-игра открытия наборов для бренда **«Золотое Яблоко»**.
UI стилистически соответствует goldapple.ru: кремовый фон,
жирный гротеск **Unbounded**, body-шрифт **Inter**, сдержанные
акценты рарности, скруглённые кнопки-пилюли.

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

## Структура

```
main.py                   — точка входа
core/                     — логика: паки, вероятности, дубликаты, сохранение
ui/                       — PyQt6 экраны: меню, выбор пака, открытие, коллекция, скидки
animation/                — анимации: эффекты, хореография по типу пака
assets/
    fonts/                — Unbounded, Inter (bundled TTF)
    cards/                — сгенерированные карточки по рарности
    packs/                — обложки паков
    sounds/               — звуки (опционально)
data/                     — JSON базы: карты, вероятности, сохранение
utils/
    config.py             — все константы
    theme.py              — фирменный стиль (цвета, шрифты, QSS)
    asset_loader.py       — автозагрузка текстур
    logger.py             — логгирование
product_illustrations.py  — векторные иллюстрации товаров (PIL)
generate_cards.py         — сборщик: иллюстрации + композиция + рамки + экспорт
```

## Фирменный стиль

- **Шрифты**: Unbounded (заголовки) + Inter (body), TTF встроены с полной
  поддержкой кириллицы и латиницы
- **Основной фон**: кремовый `#F7F5F0`
- **Текст**: чёрный `#0F0F0F`
- **Акценты**: золотой `#D4AF37`, фирменный зелёный `#9FE85A`
- **Рарности**: серо-бежевый → синий → пурпур → золото

## Иллюстрации товаров и эффекты рарностей

Каждая из 50 карточек использует собственную векторную иллюстрацию,
нарисованную средствами PIL в `product_illustrations.py`: бутылочки,
тюбики, баночки, палетки, помады, парфюмы, фен, выпрямитель, наборы
кистей, бьюти-бокс и т.д. У каждого товара — мягкая тень, блик,
этикетка с акцентной полоской.

Зона иллюстрации оформляется по рарности:
- **Common** — чистая кремовая подложка
- **Rare** — голубоватый градиент + тонкая сетка точек
- **Epic** — лавандовый градиент + редкие пурпурные блёстки и звёздочки
- **Legendary** — золотой градиент + частицы золота + двойная золотая
  foil-рамка по периметру карточки

Под каждой карточкой — индикатор тира (1–4 точки + подпись `tier N`)
и скидка крупным Unbounded Black.

## Перегенерация карточек

Если нужно пересоздать изображения (например, после редактирования
`data/cards.json`):

```bash
python generate_cards.py
```

## Игровой процесс

1. Главное меню → «Открыть набор»
2. Выбор пака (Common / Epic / Legendary)
3. Анимация открытия + последовательное раскрытие карт
4. 3 одинаковые карты → автоматическая скидка
5. Прогресс сохраняется в `data/player_save.json`
